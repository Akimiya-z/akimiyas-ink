这篇文章用来记录这个网站第一版是怎么搭出来的。

## 这次做了什么

从主题模板拉下来之后，先做的是“减法”：

- 去掉 `docs`
- 去掉 `projects`
- 去掉 `links`
- 关闭默认评论系统

这样站点结构会更像一个个人主页加博客，而不是主题演示站。

## 为什么选 GitHub + Vercel

这套组合对新手最友好：

- 代码放在 GitHub，后续好管理
- Vercel 免费额度足够个人站使用
- Astro 部署配置很省事

最重要的是，这个主题默认更适合直接部署到 Vercel，而不是折腾 GitHub Pages 的子路径兼容。

## 下一步

这个站点接下来还会继续改：

- 把占位头像换成自己的头像
- 把站点地址换成真实的 Vercel 域名或自定义域名
- 持续补充真正的博客内容
